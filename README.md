# html-documentation
### Block level elements
* All heading tags (`h1 - h6`)
* Divisions (`div`)
* Lists
  * Ordered (`ol`)
  * Un-ordered (`ul`)
   * List-style-type:
    * circle
    * disc (`default`)
    * square
    * none
    Example
```html
      <ul style="list-style-type:none">
			      <li> Coffee </li>
			      <li> Tea </li>
			      <li> Milk </li>
		    </ul>
```
  * description list (`dl`)
    * term (`dt`)
    * descriptive data (`dd`)
  * list-item (`li`)
